# Alish Hamilton Wellness Theater

A Hamilton-inspired medicine and wellness tracker for Alish.

## Included
- Hamilton theater-style landing page
- Medicine tracker with local saving
- Mood check-in
- Mood-to-Hamilton YouTube matching
- Hamilton quote card
- Wellness score
- Favorites section
- Daily encouragement
- Mockup reference image and cropped image assets

## How to Use
Open index.html in a browser.

## GitHub Pages
Upload the contents inside this folder to your GitHub repo root.
Then enable Pages:
Settings → Pages → Deploy from branch → main → /root

## Note
No copyrighted Hamilton music is bundled. The music buttons open YouTube searches.
